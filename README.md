## Notable source-corrections

This map intentionally corrects things the prose docs get wrong:

- **20 packages**, not the ~12 the docs list — adds `vue`, `react-native`, `a2ui-renderer`, `agentcore-runner`, `demo-agents`, and the config packages.
- **There is no `packages/agent`** — the `BuiltInAgent` (Vercel AI SDK) lives in `packages/runtime/src/agent/`.
- **`runtime` is dual-architecture** — a legacy GraphQL **V1** layer (+ 9 LLM service adapters) and a modern fetch-based **V2** layer (`CopilotRuntime`, `AgentRunner`, endpoints).
- **`@copilotkitnext/{react,agent,runtime}`** are externally-published packages (consumed by `showcase/shell` via the `next` dist-tag); only **`@copilotkitnext/angular`** is built from this repo.

## Regenerating

The vault was generated by scripts in `.build/` (a dot-folder, hidden inside Obsidian):

```bash
node .build/add-tags.js              # nested #layer/#type/#pkg tags from frontmatter
node .build/add-overview-aliases.js  # aliases so scoped/`/`-containing [[links]] resolve in Obsidian
node .build/make-canvas.js           # rebuild the 4 Canvas boards + graph color groups
node .build/verify.js                # link / orphan / canvas integrity check
```

`.build/conventions.md` is the authoring contract (note format, frontmatter schema, linking rules, and the source-corrections above).
