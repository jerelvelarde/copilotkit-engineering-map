# Examples - v2 docs

**Framework:** A **Mintlify** documentation site (`@copilotkit/docs`), not a runnable app. `npm run dev` runs `mintlify dev --port 4000`; content is `.mdx` (`introduction.mdx`, `reference/`) configured by `docs.json`. Only runtime dep is `sharp` (image processing).

**Demonstrates:** Reference/getting-started docs that live alongside the examples (distinct from the primary Fumadocs site documented under the [[Docs-Site MOC]]). Included in `examples/v2/` for colocation with the V2 starters.

**CopilotKit packages:** none (documentation content only).

Part of [[Examples - v2 starters]]. For the main docs site see the [[Docs-Site MOC]].
