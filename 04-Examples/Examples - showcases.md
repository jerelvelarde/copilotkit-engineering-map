# Examples - showcases

Category note for `examples/showcases/` — full-featured demo applications that highlight [[@copilotkit/react-core]] / [[@copilotkit/react-ui]] / [[@copilotkit/runtime]] capabilities in real-world scenarios. These are consolidated demo repos (not workspace members): nearly all pin **published** `@copilotkit/*` versions rather than `workspace:*`, and most live as either a single Next.js app or a pnpm/Turbo monorepo with a `frontend/` (or `apps/web`) UI plus an `agent/` backend.

Counterpart categories: [[Examples - v1 legacy]], [[Examples - v2 starters]]. (Other categories — `canvas`, `integrations`, `e2e` — are owned elsewhere; see the [[Examples MOC]].)

## What these demonstrate
- **Generative UI** in all three flavours ([[A2UI (Generative UI)]], MCP Apps, static/controlled) — see [[Examples - generative-ui]], [[Examples - generative-ui-playground]], [[Examples - mcp-apps (showcase)]].
- **Agent-Native Applications (ANA)** — shared state + Human-in-the-Loop research/canvas apps — [[Examples - research-canvas (showcase)]], [[Examples - deep-agents]], [[Examples - multi-agent-canvas]].
- **Multi-framework agents over [[AG-UI Protocol]]** — LangGraph (JS & Python), Google ADK, Mastra, PydanticAI, Strands, Microsoft Agent Framework (.NET), CrewAI.
- **Real-app patterns** — banking/authorization, kanban, spreadsheet, presentation, todo, dashboards.
