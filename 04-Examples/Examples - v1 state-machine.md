# Examples - v1 state-machine

**Framework:** Next.js 14 (React 18) workspace example (`@copilotkit-examples/state-machine`), `workspace:*` deps. Uses `reactflow` to visualize the graph.

**Demonstrates:** A **state-machine copilot** — a car-sales flow modeled as explicit states with contextual transitions, showing how to build sophisticated multi-stage conversations where the copilot's available actions change per stage. Uses [[@copilotkit/runtime-client-gql]] for the V1 transport.

**CopilotKit packages:** [[@copilotkit/react-core]], [[@copilotkit/react-ui]], [[@copilotkit/runtime]], [[@copilotkit/runtime-client-gql]] (`workspace:*`).

Part of [[Examples - v1 legacy]].
