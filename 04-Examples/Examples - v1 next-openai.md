# Examples - v1 next-openai

**Framework:** Next.js 15 (React 19 RC) workspace example (`next-openai`), `workspace:*` deps. The fullest V1 reference app — it depends on nearly every V1 package.

**Demonstrates:** A broad Next.js + OpenAI integration exercising the complete V1 stack: chat UI, AI-autosuggest text entry via [[@copilotkit/react-textarea]] ([[react-textarea - CopilotTextarea]]), readable context, frontend actions, and the [[runtime - GraphQL Layer (v1)]] runtime through [[@copilotkit/runtime-client-gql]]. Often used as the canonical end-to-end V1 smoke app. Uses `langchain` and `react-markdown`.

**CopilotKit packages:** [[@copilotkit/react-core]], [[@copilotkit/react-ui]], [[@copilotkit/react-textarea]], [[@copilotkit/runtime]], [[@copilotkit/runtime-client-gql]], [[@copilotkit/shared]] (all `workspace:*`).

Part of [[Examples - v1 legacy]].
