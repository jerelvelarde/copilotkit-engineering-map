# Examples - generative-ui

**Framework:** None runnable — this directory is **documentation + assets only** (`README.md` + `assets/`), no `package.json`.

**Demonstrates:** A landing/explainer for **Generative UI for Agentic Apps**. It defines the three types of generative UI and points at the runnable playground:
- **Controlled Generative UI** — driven by [[AG-UI Protocol]] (agent emits tool calls, frontend renders).
- **Declarative Generative UI** — [[A2UI (Generative UI)]] + Open-JSON-UI (agent emits a UI spec).
- **Open-ended Generative UI** — MCP Apps (HTML/JS in sandboxed iframes).

**CopilotKit packages:** none directly (reference content). The runnable counterpart is [[Examples - generative-ui-playground]].

Part of [[Examples - showcases]].
