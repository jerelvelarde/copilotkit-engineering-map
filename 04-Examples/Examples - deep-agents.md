# Examples - deep-agents

**Framework:** Next.js 16 (React 19) single app (`react-markdown` for file rendering). Package name `deep-research-assistant`.

**Demonstrates:** A **Deep Agents** research assistant showing the three pillars of the LangGraph DeepAgents pattern — **planning (todos)** with status indicators, **memory/files** (agent-authored markdown viewable + downloadable), and **generative UI** (rich tool-call rendering with expandable details). Uses [Tavily](https://www.tavily.com/) for live web search.

**CopilotKit packages:** [[@copilotkit/react-core]], [[@copilotkit/react-ui]], [[@copilotkit/runtime]] (pinned `^1.51.0`).

Part of [[Examples - showcases]]. See also the variants [[Examples - deep-agents-finance-erp]] and [[Examples - deep-agents-job-search]].
