# Examples - v1 travel

**Framework:** Next.js 14 (React 18) workspace example (`@copilotkit-examples/travel`), `workspace:*` deps. `react-leaflet` maps + extensive Radix UI.

**Demonstrates:** An AI **travel planner** — create, manage, and explore trips with real-time map visualization (Leaflet / Google Maps). The copilot manipulates trip state and the map view, a V1 frontend-action + readable-context showcase.

**CopilotKit packages:** [[@copilotkit/react-core]], [[@copilotkit/react-ui]], [[@copilotkit/runtime]] (`workspace:*`).

Part of [[Examples - v1 legacy]]. Distinct from the multi-agent [[Examples - a2a-travel]] showcase.
