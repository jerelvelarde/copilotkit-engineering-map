# Examples - todo (showcase)

**Framework:** Next.js 14 (React 18) single app. Package name `todo`. Notably pins the earliest versions in the showcase set: `@copilotkit/*@1.0.0-beta.2`.

**Demonstrates:** The canonical **simple todo app** — the minimal "hello world" of CopilotKit, where the copilot adds/edits/completes todos (`nanoid` for ids). A good first read for the basic frontend-action wiring.

**CopilotKit packages:** [[@copilotkit/react-core]], [[@copilotkit/react-ui]], [[@copilotkit/react-textarea]], [[@copilotkit/runtime]], [[@copilotkit/shared]] (pinned `1.0.0-beta.2`).

Part of [[Examples - showcases]]. For agent-backed todo boards see [[Examples - pydantic-ai-todos]].
