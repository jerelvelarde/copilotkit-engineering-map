# Examples - v1 node-express

**Framework:** Backend-only **Express** workspace example (`node-express`), `workspace:*` deps. No frontend.

**Demonstrates:** Hosting the **V1 CopilotKit runtime on Express** — wiring the [[runtime - GraphQL Layer (v1)]] endpoint via the Express integration from [[runtime - Framework Integrations (v1)]]. A minimal server-side reference for non-Next.js deployments.

**CopilotKit packages:** [[@copilotkit/runtime]], [[@copilotkit/shared]] (`workspace:*`). Plus `express`.

Part of [[Examples - v1 legacy]]. The V2 equivalent is [[Examples - v2 node-express]].
