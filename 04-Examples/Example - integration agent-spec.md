# Example - integration agent-spec

Starter for building agents with **Agent Spec** and CopilotKit, featuring **A2UI-powered frontend tool rendering**. Part of [[Examples - integrations]].

- **Framework:** Agent Spec, Python (FastAPI), uv. Agent under `src/` (`pyproject.toml` + `uv.lock`), OpenAI-compatible LLM.
- **Demonstrates:** A2UI generative UI tool rendering — calendar, inbox, email compose, and a daily-brief dashboard — generated by the agent and rendered client-side. Strong example of [[A2UI (Generative UI)]] and [[Tools (Frontend & Backend)]] over the [[AG-UI Protocol]].
- **CopilotKit packages:** the widest set in the integrations folder — [[@copilotkit/react-core]], [[@copilotkit/react-ui]], [[@copilotkit/runtime]], [[@copilotkit/runtime-client-gql]], [[@copilotkit/shared]], [[@copilotkit/a2ui-renderer]] (pinned `1.57.1`). A2UI plumbing via `@ag-ui/a2ui-middleware`, `@ag-ui/client@0.0.52`, `@ag-ui/core`, `@ag-ui/encoder`, `@ag-ui/proto`.
- **Frontend:** Next.js App Router (`src/`).
