# Examples - v2 react-native

**Framework:** A bare **React Native** app under `demo/` (`@copilotkit/react-native-demo`, `react-native@0.85`, native `android/` + `ios/` projects; CLI scripts `react-native run-ios/android`, Jest tests). Not Expo. CopilotKit deps `workspace:*`.

**Demonstrates:** The reference integration of [[@copilotkit/react-native]] on a real device/simulator — the RN-safe subset of the React stack ([[react-core - headless export (RN)]]) plus the native polyfills/streaming-fetch the package provides.

**CopilotKit packages:** [[@copilotkit/core]], [[@copilotkit/react-core]], [[@copilotkit/react-native]], [[@copilotkit/shared]] (all `workspace:*`). Plus `react-native`, `react-native-safe-area-context`.
