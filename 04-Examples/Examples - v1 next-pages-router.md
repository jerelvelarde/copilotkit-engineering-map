# Examples - v1 next-pages-router

**Framework:** Next.js 14 (React 18) **Pages Router** workspace example (`next-pages-router`), `workspace:*` deps.

**Demonstrates:** The V1 runtime mounted under the **Next.js Pages Router** API (vs. the App Router) — verifying [[runtime - Framework Integrations (v1)]] for the pages-router handler. Otherwise a standard V1 chat + textarea app.

**CopilotKit packages:** [[@copilotkit/react-core]], [[@copilotkit/react-ui]], [[@copilotkit/react-textarea]], [[@copilotkit/runtime]], [[@copilotkit/shared]] (`workspace:*`).
