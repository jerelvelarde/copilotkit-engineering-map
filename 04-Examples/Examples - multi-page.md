# Examples - multi-page

**Framework:** **Remix** (React Router) app on Vite (`my-remix-app`, React 18, `@remix-run/*` 2.x). No README.

**Demonstrates:** Using CopilotKit across a **multi-page Remix app** — a contacts/CRUD-style sample (`match-sorter`, `sort-by`) showing the copilot persisting across route navigation rather than a single-page surface. Notably runtime-less in its deps (frontend hooks + UI only).

**CopilotKit packages:** [[@copilotkit/react-core]], [[@copilotkit/react-ui]] (pinned `^1.9.1`).

Part of [[Examples - showcases]]. For the V2 React-Router runtime starter, see [[Examples - v2 react-router]].
